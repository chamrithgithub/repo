Instructions for locally setting up the website.
Open Wamp server or Xamp server.
In Wamp server file(most of the time in C drive)there is folder called www.
Open the www folder.
Copy the index.html,file.php and credentials.txt file into the www folder.
Open up the browser and type http://localhost:8080/index.html.
Then the phishing website will be opened.
